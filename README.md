# AccessiChat WebApp — Android APK

A native Android WebView wrapper for the AccessiChat web application, designed specifically for blind and visually impaired users.

## 🌐 Web App
https://access-chat-connect.base44.app

## 📱 Features
- Full WebView wrapper loading AccessiChat
- All Android permissions granted (microphone, camera, storage, location, contacts, Bluetooth)
- WebRTC support for voice and video calls
- Geolocation support
- File upload support
- Pull-to-refresh
- Offline error page
- TalkBack and accessibility-friendly

## 🔐 Permissions Requested
- INTERNET
- CAMERA
- RECORD_AUDIO
- ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION
- READ/WRITE EXTERNAL STORAGE
- READ_MEDIA_IMAGES / READ_MEDIA_VIDEO / READ_MEDIA_AUDIO
- READ_CONTACTS / WRITE_CONTACTS
- READ_PHONE_STATE
- POST_NOTIFICATIONS
- BLUETOOTH / BLUETOOTH_CONNECT
- VIBRATE / WAKE_LOCK

## 📲 Installation
1. Download `AccessiChatWebApp_v1.apk`
2. On your Android phone, go to Settings → Security → Allow unknown sources
3. Tap the APK file to install
4. Open AccessiChat and log in

## 🛠 Build Requirements
- Android Studio or Gradle 8.4
- Java 17
- Android SDK 34
- minSdk 21 (Android 5.0+)
